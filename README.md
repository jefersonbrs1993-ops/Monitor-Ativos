# 🚀 Monitor-Ativos

Sistema de monitoramento de ativos desenvolvido com *Python* e *FastAPI*. O projeto foca em fornecer uma interface rápida e eficiente para o gerenciamento e acompanhamento de status de dispositivos ou serviços.

## 🛠️ Tecnologias Utilizadas
* [Python 3.10+](https://www.python.org/)
* [FastAPI](https://fastapi.tiangolo.com/) - Framework web moderno e rápido.
* [SQLite](https://www.sqlite.org/index.html) - Banco de dados leve para armazenamento local.
* [Uvicorn](https://www.uvicorn.org/) - Servidor ASGI de alta performance.

## 📂 Estrutura do Projeto
* app/: Contém a lógica principal da API e conexão com banco de dados.
* requirements.txt: Lista de dependências do projeto.
